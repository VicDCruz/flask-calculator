# flask-calculator
Easy API calculator to know about microservices

I use a `dockerfile` to create this env

# Steps
1. Build this `dockerfile` image
2. `docker run --rm -d  -p 5000:5000/tcp -v $(pwd):/calculator flaskcalculator:latest`
